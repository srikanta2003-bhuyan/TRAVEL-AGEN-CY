🌍 Holiday Planner Website

A modern and responsive **Travel Agency Website** built using **HTML, CSS, and JavaScript**. This project showcases a clean UI/UX design with multiple sections like destinations, packages, testimonials, and more.

🚀 Features

✅ Fully responsive design (Mobile, Tablet, Desktop)
✅ Modern and clean UI
✅ Navigation bar with smooth scrolling
✅ Services / Features section
✅ Top Destinations showcase
✅ Featured travel packages
✅ Client testimonials slider
✅ Newsletter subscription form
✅ Footer with useful links and social icons

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- [Remix Icon](https://remixicon.com/)
- [Swiper.js](https://swiperjs.com/) (for sliders)
- [ScrollReveal](https://scrollrevealjs.org/) (for animations)
